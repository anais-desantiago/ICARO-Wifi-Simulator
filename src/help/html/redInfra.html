Creación de red de Infraestructura

Para crear la red podrá hacer lo siguiente:

  • Doble clic sobre el AP o el enrutador segú sea el caso.
  • Haciendo clic derecho sobre el AP o el enrutador segú sea el caso y escoger la opciún Configurar.

Después de haber realizado lo anterior, deberá realizar la configuración de la red:

  • Configuración LAN
    1. IP
    2. Máscara: las máscaras permitidas por la herramienta son aquellas establecidas según la clase de la dirección IP, no se permiten máscaras de subred.
    3. Escoger si se asignarán direcciones por DHCP o no.

      Si escoge que se envíen direcciones por DHCP, deberá indicar la cantidad de direcciones a enviar.

  • Configuración WLAN.
    1. SSID
    2. Modo. Si aplica, escoger la banda.
    3. Región.
    4. Canal.
    5. Tipo de Seguridad.

      Si escoge que haya seguridad, bien sea WEP o WPA, deberá escribir la contraseña.

  • Configuración WAN. (Si el nodo es un enrutador)

    Seleccione el tipo de conexión.

    Si escogió IP estática:

    1. IP WAN
    2. Máscara: las máscaras permitidas por la herramienta son aquellas establecidas según la clase de la dirección IP, no se permiten máscaras de subred.
    3. Puerta de enlace predeterminada. Generalmente es la IP WAN pero el último octeto es .254
    4. DNS Primario.
    5. DNS Secundario.
  • Guardar
