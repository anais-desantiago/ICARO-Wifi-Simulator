Ventana principal

ICARO Wi-Fi Simulator se divide en cuatro secciones:

  • La barra de menú: esta barra permite seleccionar las distintas funciones que realizará ICARO Wi-Fi Simulator. Están presentes cinco menús.
    • Archivo: permite agregar una nueva área de trabajo, guardar, cargar, cerrar.
    • Recurso: establece el tipo de red a utilizar (Ad-hoc o Infraestructura) y añadir componentes según el tipo de red.
    • Simulador: permite iniciar, detener, guardar y cargar la simulación.
    • Ayuda: el usuario podrá consultar el uso de la herramienta y mayor información sobre ella en la secciún Ayuda.
  • El panel de botones: contiene accesos directos a funciones pertenecientes a la barra de menú, tales como agregar componentes e iniciar y detener la simulación.
  • El panel central es el área de trabajo en el que se agregarán los diversos componentes, y se podrán mover y configurar.
  • El panel inferior es el área de consola en el que se podrá mostrar la información de cada uno de los componentes según se seleccione, además se mostrará otra información relevante como el resultado de ping.
